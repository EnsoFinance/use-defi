# usePositions

Fetches all available positions on enso.finance, that can be run as a shortcut.

## Usage

```tsx
import { usePositions } from 'use-defi'

export function Demo() {
  const availablePositions = usePositions()

  return (
    <div>
      <div>{availablePositions}</div>
    </div>
  )
}
```
